Web Development Journey: From HTML to WordPress Plugins 🚀
Welcome to my web development portfolio repository! As a final-year BSCS student, I created this space to document my 6-week progression through my web development bootcamp, transitioning from static front-end foundations to dynamic server-side programming and WordPress plugin development.

📖 About This Repository
This repository contains all the projects, exercises, and code snippets I developed during the course. Each week focuses on a new layer of the web stack, ultimately culminating in the creation of custom WordPress themes and plugins.

🛠️ Technologies & Tools Learned
Front-End: HTML5, CSS3 (Flexbox, Responsive Design), JavaScript (ES6+, DOM Manipulation)

Back-End: PHP, MySQL (Basics)

CMS & Ecosystem: WordPress Core, Theme Development, Plugin API (Hooks, Shortcodes, Options API)

📅 Course Outline & Project Milestones
Week 1: Web Foundations & Semantic HTML
Understanding the mechanics of the web (Client-Server, HTTP/DNS) and building structured web pages.

Concepts: DOM structure, Semantic HTML5 tags (<header>, <main>, <article>, etc.), Links, Media, and File paths.

Project: Built a multi-page personal profile website (About Me, Hobbies, Contact) using strictly semantic HTML.

Week 2: Styling with CSS & Responsive Design
Controlling visual presentation and ensuring layouts look great across all devices.

Concepts: The Cascade, Specificity, Box Model, Flexbox (display: flex, justify-content, align-items), and Media Queries.

Project: Styled the Week 1 HTML project, implementing a fully responsive navigation bar using Flexbox for both desktop and mobile views.

Week 3: Front-End Interactivity with JavaScript
Adding dynamic behavior and interactive elements to the DOM.

Concepts: JS Fundamentals (Variables, Functions, Control Flow), DOM Manipulation, and Event Handling (addEventListener, preventDefault).

Project: Added interactive features to the styled site, including a Dark Mode toggle, an image slider, and client-side form validation.

Week 4: Server-Side Power with PHP
Transitioning to the back-end to process data and generate dynamic content.

Concepts: PHP Basics, Superglobals ($_GET, $_POST), Control Structures, Form Handling, and an introduction to MySQL/Prepared Statements.

Project: Developed a functional contact form that securely sends data to a PHP script, validates the input, and returns a dynamic confirmation message.

Week 5: WordPress Theme Development & The Loop
Converting a static HTML/CSS template into a fully functional, dynamic WordPress theme.

Concepts: Local Server Setup, WordPress Template Hierarchy, The Loop, Core Files (functions.php, header.php, etc.), and Template Tags.

Project: "WordPress-ified" the static website by breaking it into dynamic PHP components and using The Loop to pull in dynamic blog posts from the WP Admin dashboard.

Week 6: WordPress Plugin Development
Extending core WordPress functionality by building custom plugins from scratch.

Concepts: Plugin Anatomy, Actions vs. Filters, Shortcode API, Admin Menus, Security (Nonces, Sanitization, Escaping), and the Options API.

Project: Built a "Custom Welcome Message" plugin featuring:

An Action Hook to inject a custom message into the site header.

A custom [display_date] shortcode.

A dedicated options page in the WP Admin dashboard for the site owner to update the welcome text dynamically.

🚀 How to Navigate This Repo
Navigate to the respective Week-# folders to view the source code for each module.

For PHP and WordPress projects (Weeks 4-6), a local server environment (like Local by Flywheel or XAMPP) is required to run the code.
